# DreamJournal

A site that lets you document your dreams.

---

## Features

- Add and save your dreams with details like title, description, date, location, and tags.
- View a list of all your recorded dreams.
- Store dreams persistently in a JSON file on the backend.
- Simple and clean interface to quickly log and browse dreams.

---

## Tech Stack

- Frontend: HTML, CSS, JavaScript (or React, if applicable)
- Backend: Node.js with native HTTP module or Express.js
- Data storage: JSON file on the server
